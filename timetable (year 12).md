# Year 12 Mock A* Timetable

## Context assumptions
- 3 A-levels, school ~8:30–3:30
- Mocks: January (roughly 16 weeks from September start)
- Goal: efficient, not just long hours

## Phase Breakdown (16 weeks)

### Phase 1 — Weeks 1–6 (Sep–mid Oct): Build foundations
- Keep up with every lesson — annotate notes same evening
- Make condensed notes / flashcards as you go (Anki or physical)
- No past papers yet — too early, content incomplete
- Each subject: 2x per week, rotate daily

### Phase 2 — Weeks 7–10 (mid Oct–Nov): Active recall kicks in
- Switch from note-making → flashcard drilling (20 min/day per subject)
- Start topic-by-topic past paper questions (not full papers)
- Mark your own work against mark schemes — this is the highest-value habit
- Identify weak topics → double session on those that week

### Phase 3 — Weeks 11–14 (Nov–Dec): Past paper mode
- Full past papers under timed conditions — 1 per subject per week minimum
- Mark → categorise errors → revisit those specific topics
- Build an error log (spreadsheet or notebook): topic, mistake type, fix
- Reduce new note-making; focus is consolidation

### Phase 4 — Weeks 15–16 (Jan): Mock sprint
- Past papers daily, rotating subjects
- Review error log — only revise things you've gotten wrong before
- Sleep 8h minimum — consolidation is neurological
- Stop creating new resources; use only what you've built

---

## Non-negotiables for A*

| Habit | Why |
|-------|-----|
| **Past papers from Week 7** | Mark schemes teach examiner language — the biggest gap between B and A* |
| **Error log** | Prevents repeating the same mistakes |
| **Sleep 8h** | Memory consolidation — non-negotiable |
| **No passive re-reading** | Highlighting/re-reading has near-zero retention; flashcards + retrieval practice do |
| **One full rest day** | Burnout in November is the most common reason for poor January mocks |
| **Teach-back method** | Explain each topic aloud as if teaching — exposes gaps immediately |

---

## Subject-specific note
If one of your three subjects is Chemistry, spaced repetition on equations/mechanisms from Week 1 pays the highest dividend — don't leave organic mechanisms to Phase 3.
